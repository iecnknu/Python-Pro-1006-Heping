format( )函式 - 格式操作符號

format( )函式 - 格式操作符號

🗒️格式化輸出 - 格式操作符號(格式限定符號)

在【{ }】中，使用【:】冒號作為呼叫，在其後加入格式操作符號，讓參數以特殊格式輸出。
如果【:】冒號後沒有加入格式操作符號，會以預設的空白填充。

'{:格式操作符號}'.format(參數1)

格式操作符號格式： [ [ fill ] align ] [ sign ] [ # ] [ 0 ] [ width ] [ ,] [ .precision ] [ type ]
撰寫程式時不需要【 [ 中括號】，且如果一次要使用多個操作符號，請一定要按照上述的順序填寫。

名稱 功能
fill 填滿空位字元
align 設定對齊方式
sign 設定正負號
# 設定顯示進位制
0 以0填滿 正負號 和 數字 之間的空位
width 設定輸出的寬度
, 設定千位符號
.precision 設定小數點的位數
type 設定數值的表示法
📝輸出的寬度 - width

通常會搭配【填滿空位字元 - fill】和【對齊方式 - align】。

填滿空位的預設值為空白。
對齊方式預設靠右。
如果要顯示的字串超過 設定的寬度，則寬度會自動增加。

'{:填滿空位字元 對齊方式 輸出的寬度}'.format(參數1)
📝對齊方式符號 - align
> 靠右對齊
< 靠左對齊
^ (Shift + 6) 置中對齊
👨‍💻範例：

格式化輸出課堂範例-2

#輸出寬度10位的字串


#輸出寬度10位，並靠右對齊的字串


#輸出寬度10位，並以@填滿空白
📝千位符號 - ,

輸出的數值會加上千分位符號，在一些統計或金融系統之中相當好用。

'{:,}'.format('1000')
📝正負號 - sign

在數值前加上 + 、 - 符號。

'{:+}'.format('1000')
📝數值表示法 - type
b 二進位
d 十進位
o 八進位
x 十六進位
f、F 浮點數(預設精確到小數點後6位)
e、E 科學記號
n 數值
s 字串
% 百分比
👨‍💻範例：

格式化輸出課堂範例-3

#以浮點數表示16


#以百分比顯示


#以二進位表示8
📝小數點的位數 - .precision

設定小數點後的位數，通常會搭配【輸出的寬度 - width】和【數值表示法 - type】。
如果小數點後要顯示的位數，小於原本數值擁有的位數，則四捨五入後一位來顯示。

例：1.27，小數點後有兩位，但如果我們只要求顯示小數點後一位，則7會4四捨五入進位，顯示為1.3。

'{:輸出的寬度 小數點的位數 數值表示法}'.format(參數1)
👨‍💻範例：

格式化輸出課堂範例-4

number = 9527.578

#顯示寬度為5，數值表示法為浮點數，精準到小數點後2位


#顯示寬度為8，數值表示法為科學記號，精準到小數點後4位


#顯示寬度為10，數值表示方式為百分比，精準到小數點後3位
👓補充：科學記號

科學記號是一種數學的表達方式，通常用在工程或電腦的浮點數(floating point)。
前面稱為有效數值 (mantissa)後面是指數(exponent) 。
E 代表10的次方數，
1000的表示方式為1x103，機器的表示法為1E+03。
0.001的表示方式為1x10-3，機器的表示法為1E-03。

科學記號的優點是能清晰地表示數值的數量級，而不會被一堆0搞得眼花繚亂。
一個質子的質量為：0.00000000000000000000000167262158公克，
以科學記號表示的話：1.67262158x10-24，機器的表示法為1.67262158x10E-24
