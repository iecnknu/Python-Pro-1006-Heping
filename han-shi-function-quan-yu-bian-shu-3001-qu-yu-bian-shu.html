函式 Function - 全域變數、區域變數

  1. 全域變數、區域變數
  2. 關鍵字 global

全域變數、區域變數

經過函式參數的洗禮，會發現在某些執行主體中會無法存取一部分的變數。

取決於變數的有效範圍，我們將之細分為 全域變數( global variable )與 區域變數( local variable )。

  • 全域變數：在程式碼所有的範圍中，皆能存取全域變數的值。
  • 區域變數：被定義在函式之中、只存在於函式內的變數，所以只能在函式的執行主體之中存取區域變數的值。
範例：

全域區域變數範例-1 / 解答

#請問會印出什麼值？

1. def example():
2.     print(text)
3.
4. text = 'HELLO'
5. example()

全域區域變數範例-2 / 解答

#請問會印出什麼的值？

1. def example():
2.     text = 'hello'
3.
4. text = 'HELLO'
5. example()
6. print(text)

全域區域變數範例-3 / 解答

#請問會印出什麼值？

1. def example():
2.     print(text)
3.     text = 'hello'
4.
5. text = 'HELLO'
6. example()
7. print(text)

關鍵字 global

如果想要在函式範圍內、修改全域變數的值時，就要使用【關鍵字global】，告訴Python我們要參考全域變數，而不是像全域區域變數範例-2 ，在函式範圍之中又創立了一個相同名稱的區域變數。

def example():
    global 變數

變數 = 'a'
example()
課堂練習：

關鍵字global練習-1

#請修改以下程式，讓example()函式之中的【text變數】能成為全域變數，最後印出小寫的hello！

1. def example():
2.     text = 'hello'
3.
4. text = 'HELLO'
5. example()
6. print(text)
