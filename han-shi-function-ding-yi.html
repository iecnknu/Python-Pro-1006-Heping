什麼是函式？

首先複習一下，在入門教材中我們介紹的函式與方法。

  • 我們通常將一個常利用到的小功能獨立寫成一個「函式」，並通過名字來調用函式，使程式資源可以有效被運用。例如：
print("abc")
  • 方法是一個被定義在類別裡的函式，以物件本身為預設參數，通過名字來調用方法。例如：
list1 = [1,2,3,4,5]
list1.append(6)

以上為Python為一些基礎的類別內建的函式庫，它方便我們在使用相同的功能時，不用重複撰寫程式碼。

而善加使用函式，不管是閱讀或是修改程式碼，對程式設計師來說可讀性都會大大的提高。

最重要的就是我們能將軟體的"流程"和"功能"分開書寫，簡化程式的複雜度。

定義函式

Python 已經內建了許多非常好用的函式，比方說印出內容值的print( )、讓使用者輸入的input( )、顯示資料型態的type( )或是讓資料可以轉型的int( )及str( )。

不過當我們有特殊需求時，就可以依照Python定義函式的規則，寫出自己的函式：

def 函示名稱():
    執行主體
名稱 用法
def 表示即將定義函式，空一個空白(space)，接上一個獨一無二的函式名稱和一對小括號，並以冒號將之與執行主體區隔。
函示名稱 函式命名規則跟變數命名規則相同。
執行主體 撰寫要執行的程式碼。以 def 關鍵字為基準，以4個空白或1個tab 鍵向右縮排( 兩者不可混用 )。

呼叫函式

Python為直譯式程式語言，直譯器會從第1行依序直譯處理，

函式可以放置在程式碼中的任何位置，只要在呼叫前被存放在記憶體中即可。

所以慣例上，程式設計師會將函式放置在程式碼的頂端，以確保呼叫時函式已被存放在記憶體中。

呼叫函式名稱以執行函式：

def example():
    .............
    ...執行主體...
    .............

#呼叫程式名稱
example()
