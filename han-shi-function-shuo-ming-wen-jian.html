建立說明文件

Python最偉大的精神在於分享，與其人共用你設計的程式時，有一份完整的說明文件就顯得十分重要。

在定義函式時，於第二行開始，以兩組、三個單引號'''包圍對程式的說明。

由於三個單引號的特性，可以一次跨越數行撰寫說明文件。

通常說明文件至少需要這幾點，才能稱得上是有效的說明文件：

  • 該函式的用途，使用後會不會有回傳值。
  • 說明每個傳入的參數要做什麼事、型別是什麼。
  • 說明每個回傳值的意義是什麼、型別是什麼。
def example(參數1, 參數2):
    '''
    這是一個範例函式
    參數1：用途是XXX
    參數2：用途是OOO
    回傳值：回傳參數1和參數2比較的結果
    '''
    .............
    ...執行主體...
    .............

    return 回傳值

help()函式

可以使用內建的help()函式，查看函式的說明文件。

help( 函式名稱 )
