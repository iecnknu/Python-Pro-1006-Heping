定義自己的類別

定義自己的類別

在之前章節介紹過的【str字串】、【int數值】、【float浮點數】、【boolean布林值】等，都是Python物件的類別。我們可以把它們稱為【嵌入式的資料型別】。

而Python的開發者們可以自己定義專屬的【類別Class】(也被稱為型別type)，同時也可以設計該類別才有的【方法Method】和【屬性Attributes / Properties】(就是內部的變數)。

🗒️類別的基本架構
  • 使用關鍵字class作宣告，後面加上一個空格、加上你的類別名稱(通常以大寫為開頭)，最後以【: 冒號】做結尾。
  • 以一個縮排將類別的內容定義在一起。
  • 類別中的方法，和所有的方法一樣，必須以def來宣告。
  • 第一個宣告方法必須是【__init__】這個特殊方法，這是一個【初始化 initial】的方法，也就是當有人開始實例化這個類別的物件時(建構物件)，會呼叫優先尋找並呼叫這個方法，但也只會呼叫一次。在其他程式語言中，這個方法也被稱為【建構子constructor】。
  • Python類別的方法和其他程式語言不同的是，所有方法的第一個參數，都要清楚地寫出代表本身的self(也就是該物件本身)。
  • 所有的實例屬性都是伴隨著物件生成，因此每一個物件的實例屬性都是互相獨立的，當修改其中一物件的實例屬性時，並不會影響到其他實例。
  • 另一方面，類別屬性並不會隨著物件而生成，甚至不需要產生任何物件便可直接存取，每一個物件所存取的類別屬性都將是同一個，也就是說當我們修改類別屬性時，也會影響到別的實例存取。
class 類別名稱:

    #建立類別屬性

    def __init__(self, 參數....):
        ...初始化的方法...
        #在此建立實例屬性

    def 方法1名稱(self, 參數....):
        ...

    def 方法2名稱(self, 參數....):
        ...

1️⃣ 建立一個寵物Pet的類別：

class Pet:

    name = '寵物名'

    def __init__(self, name):
        print('初始化')
        self.name = name

    def hello(self):
        print(self.name + 'say hello')

    def eat(self,food):
        self.food = food
        print(self.name + ' eat ' + self.food)
  • 這個類別的建構子，也就是初始化的方法，會印出【初始化】三個字，並接受使用者傳入【參數name】、並將其值存入【屬性name】(也就是內部變數 self.name)之中。
  • hello方法並沒有接受使用者傳入參數，但執行時會印出【內部變數name】的值加上say hello。
  • eat方法接受使用者傳入的【參數food】、並將其值存入【內部變數food】。 最後印出【內部變數name】的值加上eat加上【內部變數food】的值。

2️⃣實例化這個類別：

    pet1 = Pet('Happy')
  • 建立Pet型別的實體(物件)，並傳入一個字串參數Happy。
  • 把這個物件貼上【變數名稱 pet1】的標籤。

  • Happy被存入【內部變數name】之中。

    pet1.hello()

  • 針對pet1這個Pet類別的物件，執行hello( )這個方法。

  • 印出【內部變數name】的值 + say hello

    pet1.eat('鮪魚罐頭')

  • 針對pet1這個Pet類別的物件，執行eat( )這個方法，並傳入一個字串參數鮪魚罐頭。

  • 鮪魚罐頭被存入【內部變數food】之中。

  • 印出【內部變數name】的值 + eat + 【內部變數food】的值。
