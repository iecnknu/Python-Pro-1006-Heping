XAMPP

XAMPP

🗒️免費的XAMPP

XAMPP = Apache(伺服器軟體) + MariaDB(資料庫) + PHP + Perl

XAMPP是完全免費且易於安裝的Apache發行版本，其中包含MariaDB、PHP和Perl。XAMPP開放源碼套件的設置讓安裝和使用出奇容易。

  1. 安裝XAMPP
    1️⃣除了Apache(伺服器軟體)和PHP，我們並不需要其他的服務，都不要勾選以加快安裝速度。(日後若有需要再安裝即可)
    2️⃣其餘的安裝步驟就一直Next>下去就好

  2. 執行XAMPP
    1️⃣安裝完成後，要讓Windows的防火牆允許Apache運行。
    2️⃣打開Control Panel後，點擊Apache旁的Start按鈕執行伺服器軟體。
    3️⃣最後要確定Apache有亮起綠，才代表伺服器有在運行。

  3. 放置HTML文件
    1️⃣如果剛剛沒有隨意更動XAMPP的安裝位置，應該可以在C:\路徑下找到xampp的資料夾。
    2️⃣在xampp的資料夾內尋找一個名為htdocs的資料夾，這是xampp預設的網頁資料夾。
    (直接前往路徑：C:\xampp\htdocs)
    3️⃣刪除資料夾內的所有內容，並放入自己撰寫的Google網頁。
    4️⃣每個網站資料夾的首頁檔一般名稱均設定為index，所以請將自己的Google網頁更名為：index.html
    5️⃣打開瀏覽器，前往http://localhost/，就可以看到自己的網頁了！(或者127.0.0.1)

    👀補充：
    localhost 是一個域名，這個域名對應的的IP位址是127.0.0.1，和一般的域名不同的是這個域名的DNS解析是在本地完成，不需要遠程的DNS伺服器進行解析。
    而127.0.0.1是一個回送地址，指向電腦自己本身。如果自己的電腦恰好是一台伺服器，便能模擬網路的方式進行連線。

  4. 全球資訊網！
    現在伺服器已經架設完畢了！
    只要再擁有一組固定IP，在世界上的任何人只要在瀏覽器打上你的IP位址，就能看到你的網頁了！

🗒️如果沒有固定IP或購買網址？

如果沒有特別申請，家裡固網的IP會在每次開機連上網時重新向供應商申請一個IP，我們稱之為浮動IP。

如果只是小規模、出自於興趣或純粹磨練技術的自架伺服器，開發者通常不會為此去購買網址(一年、三年繳費 + 維護費 + 資安費)，或者向網路供應商(如中華電信)申請固定網址。

但同時又想要讓全世界的人都可以欣賞你的創作該怎麼辦呢？

回頭來看這張圖：網址的作用就是要讓使用者不必背下攏長的IP，只需要借助DNS伺服器，就能找到正確的網站伺服器。
不過也需要固定IP，DNS伺服器才能起到作用。

如果主機的IP(住址)變了，那現在DNS也不知道該怎麼找了！
