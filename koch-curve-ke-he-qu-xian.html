Koch Curve 科赫曲線

Koch Curve 科赫曲線

雪花是一種奇妙、規律且美麗的曲線，其規則被【Niels Fabian Helge von Koch，海里格·馮·科赫】於1904年的論文《關於一個可由基本幾何方法構造出的，無切線的連續曲線》首次提出，人們稱之為【Koch Curve，科赫曲線】。

他推導出其公式為：
假設剛落在地面的雪花是正三角形。接著每隔一秒鐘就在正三角形的每一邊的中央三分之一處產生一個新的正三角形。

只要遵循以下步驟，就能繪製長度為x的科赫曲線：

假設一條科赫曲線長度為L：
1.畫一條 L/3 的直線。
2.左轉60°。
3.畫一條 L/3 的直線。
4.右轉120°。
5.畫一條 L/3 的直線。
6.左轉60°。
7.畫一條 L/3 的直線。

如果想要畫出一個擁有科赫曲線的正三角形，只要在畫完時再向右轉120度就可以了！
8.右轉120°。
9.1~8步驟再重複兩次。

畫出科赫曲線

Turtle模組的第四個挑戰，控制小烏龜印出科赫曲線！

創建python檔

  1. 打開sublime3， Ctrl + N 開啟一個新檔案， 緊接著Ctrl + S存檔， 將檔案命名為kochCurve-1.py， 並且將儲存目的地選在"桌面"。(方便待會兒操作)

  2. 匯入所需模組

    和所有模組一樣，首先使用import將turtle模組匯入程式之中：

    import turtle
  3. 建立場景 與 畫筆

    使用Screen()函式，建立畫布。貼上screen這個變數名稱。
    使用setup()函式，設定畫布大小為(500,500)。
    使用title()函式，設定畫布標題為'畫出科赫曲線'。
    使用Turtle()函式來建立一枝名為koch的畫筆，同時設定科赫曲線的邊長。

    同時使用mainloop()讓turtle的物件進入等待使用者操作的狀態喔！

    import turtle

screen = turtle.Screen()
screen.setup(500, 500)
screen.title('畫出科赫曲線')

koch = turtle.Turtle()
length = 240

#繪圖區

screen.mainloop()

  4. 操作指令

    接著，準備在繪圖區開始撰寫指令囉！
    依照剛剛步驟解說的1~7步

    koch.forward(length/3)
koch.left(60)
koch.forward(length/3)
koch.right(120)
koch.forward(length/3)
koch.left(60)
koch.forward(length/3)

    接著再透過迴圈，重複執行3次。
    記得每執行一次，要向右轉120度喔。

    for i in range(3):
   koch.forward(length/3)
   koch.left(60)
   koch.forward(length/3)
   koch.right(120)
   koch.forward(length/3)
   koch.left(60)
   koch.forward(length/3)

   koch.right(120)
