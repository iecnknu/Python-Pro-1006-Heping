##### 🔰第一步：安裝套件

先到【Installation of pytube】看看如何安裝吧！

![](https://drive.google.com/uc?export=download&id=1v30muiFxJTrly32V9A1WIMJZih3ncZIC)

使用下載指令安裝pytube！

```bash
pip install pytube
```

因為pytube是一個開源的套件專案，所有的程式碼都公布在GitHub上。  
有興趣得同學可以到GitHub上將完整的專案下載下來自己修改喔！
