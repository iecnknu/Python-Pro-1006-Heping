網站的骨幹 - HTML5-1

  • 什麼是HTML？
  • 定義HTML
  • 課堂練習
🗒️什麼是HTML？

  • HTML，【超文件標示語言HyperText Markup Language】。
    我們使用它來和瀏覽器溝通， 向它要求網頁該怎麼呈現、如何呈現以及呈現什麼。

  • HTML裡面包含了一系列的【元素elements】，而元素包含了【標籤tag】與【內容content】。
    【標籤tag】會定義【內容content】該怎麼呈現，比方說：

    <h1>hello</h1>
<p>hello<p>
<i>hello</i>

    用三種不同的【標籤tag】：<h1>、<p>、<i>，包圍相同的【內容content】，則會顯示三種截然不同的外貌。(點我觀看)

🗒️定義HTML

  • 打開Wikipedia或任意網頁，對著頁面點選右鍵，選擇【檢查網頁原始碼】。
    可以在第一行找到：

      <!DOCTYPE html>

  • 這一行稱為【DTD，Document Type Definition文件類型定義】。
    我們常見的瀏覽器有Microsoft的Edge、Google的Chrome和Apple的Safari，而每一家公司對html內的解析方式都有所不同，W3C為求統一標準而製定了這項規定。只要瀏覽器看到這行宣告，就必須遵守此規範去解讀 HTML 標籤，避免不相容的狀況發生。

  • HTML文件的程式碼由<html>...</html>【tag標籤】包圍，內部再分成<head>...</head>和<body>...</body>兩個不同段落。
    和Python相似的是，HTML也使用巢狀架構。
    需注意的是哪個起始標籤在最外側，它的結束標籤也必須在最外側。

      <html>
      <head>
          <title>...HTML的標題...</title>
      </head>

      <body>
          <h1>HELLO</h1>
      </body>

  </html>

  • 以上的規則讓HTML形成了一種樹結構，從根root開始，再延伸出許多枝枒。
    以樹結構表示，可以讓資料結構表答得更清楚，使用者能很輕易的了解。

  • 【標籤tag】：瀏覽器以標籤來區分該區塊的用途。每個標籤必須形成一對(起始標籤、結束標籤)，在撰寫時必須時時注意。
    【內容content】：為想要呈現的東西。
    【元素elements】：由以上兩者組成的稱之元素。

  • 基礎標籤介紹

    • 【<html>】 又被視為根元素（root element），包含了所有顯示在這個頁面上的內容。
    • 【<head> 】裡面放的是你想涵括的重要資訊，但不會顯示於網頁瀏覽者眼前的。例如，顯示於搜尋結果的關鍵字、頁面說明、CSS、字元實體集...等。有點像是Python的import一樣，會寫在HTML文件的上方。
    • 【<title>】呈現網頁的標題，通常會顯示在瀏覽器的分頁上。
    • 【<body> 】包含了所有會顯示於網頁瀏覽者眼前的內容。 無論是文字、圖片、影面、互動遊戲...等。
👨‍💻課堂練習

  1. 創建HTML檔

    打開Sublime3，
    Ctrl + N 開啟一個新檔案，
    緊接著Ctrl + S存檔，
    將檔案命名為hello.html，
    並且將儲存目的地選在"桌面"。(方便待會兒操作)

  2. 善用編輯器

    鍵入<h，會發現Sublime3很智慧地跳出標籤選項，請選擇html。
    按下enter後，編輯器瞬間自動幫我們補足HTML的檔案宣告及基本標籤，讓我們可以快速地繼續撰寫程式。

  3. 動手試試看
    請讓網頁的標題顯示為：My Website
    並讓內容顯示：Hello Python World, Hello HTML World

  4. 完成後請用瀏覽器打開，檢查是否能正常顯示。

  5. 在瀏覽器中點擊右鍵，選擇【檢查網頁原始碼】，看看是否跟你的程式碼相同？
