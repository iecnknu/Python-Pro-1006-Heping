JSON簡介

JSON簡介

🗒️JSON用途

JSON，全名是【JavaScript Object Notation】，意思是【JavaScript的物件記號法】。
【Notation - 記號法】，一種以字元表示數字或文字的資料，使用者用相同的記述方式表示之，便能開啟彼此間的溝通。
它是一種資訊交換的格式，可以穿梭在各種不同的系統之間，並且獨立於程式語言之外。

🗒️JSON物件格式

使用大括號 { 包住 名稱與值，代表是一個物件。

{ "名稱" : 值 }
{ "key" : value}

使用 雙引號 包住名稱。

使用 冒號 間隔 名稱與值。

使用 逗號 分隔多個 名稱與值。也代表後續還有資料。

值的資料型別可以是：JSON物件、數值、字串、布林值、None、串列

以JSON描述這間教室：
有40個seat，教室在1F，電腦作業系統是windows 7

{ "seat" : 40, "Floor" : "1F", "computer's OS" : "windows 7"}

稍微排列一下：

{
  "seat": 40,
  "Floor": "1F",
  "computer's OS": "windows 7"
}

teacher的值本身也是一個JSON物件，裡面包含了gender是male，height是175cm

{ "seat" : 40, "Floor" : "1F", "computer's OS" : "windows 7", "teacher" : {"gender" : "male", "height" : "175cm"}}

稍微排列一下：

{
  "seat": 40,
  "Floor": "1F",
  "computer's OS": "windows 7",
  "teacher": {
    "gender": "male",
    "height": "175cm"
  }
}

🗒️實作練習

點我前往
寫一個描述自己的JSON物件：
最外層的JSON物件名稱是person，
其值是一個JSON物件，包含了name、height、weight、head【名字、身高、體重、頭】
而head的值又是一個JSON物件，包含了hairLength、eyesColor【頭髮長度、眼睛顏色】
範例：

{"person":{"name":"Roy","height":"175","weight":"75","head":{"hairLength":"5cm","eyesColor":"Brown"}}}

排列之後：

{
  "person": {
    "name": "Roy",
    "height": "175",
    "weight": "75",
    "head": {
      "hairLength": "5cm",
      "eyesColor": "Brown"
    }
  }
}
