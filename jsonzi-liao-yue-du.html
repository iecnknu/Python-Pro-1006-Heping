資料閱讀

我們從OWM接收到的其實就是JSON格式的資訊。

有兩種做法讓它比較好閱讀：

  1. JSON editor Online
    https://jsoneditoronline.org/#/
    把剛剛收到的整段複製貼到左側的輸入框(紅色部分)，點選中間的轉換按鈕(綠色部分)，最後排版過後的JSON資料會顯示在右側(藍色部分) 。
    key值都被清楚了羅列出來，方便我們之後使用python解析。

  2. JSON Viewer
    https://chrome.google.com/webstore/detail/json-viewer/gbmdgpbipfallnflgajpaliibnhdgobh
    JSON Viewer是一款Chrome瀏覽器上的插件，當你使用Chrome開啟Json檔案時，能自動格式化。
    同樣接收到OWM的JSON格式資訊，安裝插件後，資料呈現的樣子好看多了！
