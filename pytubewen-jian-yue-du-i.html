pytube文件閱讀 - I

接下來就讓我們一個章節一個章節的從文件中取得我們需要的資訊吧！
(🤫模擬一個菜鳥工程師在使用全新套件時的思路)

  1. Installation of pytube 介紹pytube的安裝。
  2. Quickstart 教您pytube的基本用法。
  3. Working with Streams 探討各種用於處理串流的選項。
  4. Subtitle/Caption Tracks 以和查詢串流相同的方式取得字幕軌道。
  5. API 物件的詳細介紹，想要發揮套件的所有功能就必須到這裡查詢。
🔰第一步：安裝套件

先到【Installation of pytube】看看如何安裝吧！

使用下載指令安裝pytube！

pip install pytube

因為pytube是一個開源的套件專案，所有的程式碼都公布在GitHub上。
有興趣得同學可以到GitHub上將完整的專案下載下來自己修改喔！

🔰第二步：怎麼開始呢？

如果東看西看還不知如何下手，就到【Quickstart】學習最基礎的用法吧！
(😏通常一個受歡迎的第三方套件，都有可以快速指導使用者的【Quickstart】！)
