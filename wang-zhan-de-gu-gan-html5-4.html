🗒️如何送出資料給伺服器？

1️⃣請大家使用google，快速的搜尋：國立高雄師範大學資訊教育中心。
完成後請仔細觀察【URL，Uniform Resource Locator，劃一資源定位器】，也就是俗稱的【網址】。

2️⃣是否發現"國立高雄師範大學資訊教育中心"就藏在其中呢？
這其實是因為Goolge支援使用get方法來提交查詢子句，就像是OWM的請求網址一樣。
只要找出查詢的關鍵字與內容值的名稱，我們就可以自己撰寫網址提交給Goolge，而不用再透過它的搜尋輸入框。

3️⃣觀察後發現，【search? 】就是查詢的關鍵字，【 q 】則是我們的內容值的名稱。
所以我們可以將剛剛一長串的搜尋網址，拿掉我們不需要的部分，簡化成：

https://www.google.com.tw/search?q=國立高雄師範大學資訊教育中心

一樣可以完成搜尋。

4️⃣所以由此可知，Google的搜尋輸入框，基本上就是將我們輸入的文字，塞進網址之中以便向Google的伺服器要求回應相關的內容。那它又是怎麼做到的呢？

🗒️表單標籤<form>
  • <form> 用來蒐集使用者輸入的內容，並向伺服器傳輸這些數據。
<form>
    form的元素
</form>

  • <form>有幾個屬性需要設定：

    • 1️⃣action：設定當這個比單被提交時會執行的動作，通常會指向另一個網頁。
      • <form action="https://www.google.com">

    • 2️⃣method：將蒐集到的資訊傳送到伺服器的方法，有get和post兩種值。

      • get：改變URL，將蒐集到的資訊以【名稱 = 值對】的格式加到網址之中。

      • post：使用HTTP post的方法，網址不會發生變化，相對來說更為安全。

      • <form action="https://www.google.com" method="get">
🗒️輸入標籤<input>

  • 表單能夠包含<input>元素，可以設定傳入的各種形式。

  • type ：可以將【input標籤】設定成多種不同的類別，比方說文字方框、複選框、單選框、提交按鈕等等，讓網頁體驗更上一層樓。

    • text：設定只能輸入一行的文字方框
    • radio：設定單選或多選的圓形按鈕
    • submit：設定一個提交的按鈕(按下後即提交給伺服器)

  • name ：設定要傳入的內容值名稱

  • value ：設定要傳入的內容值(先設定好的就會變成預設值)

📝範例
<form action="hello.html" method="get">
  姓<br>
  <input type="text" name="firstname" value="普"><br>
  名<br>
  <input type="text" name="lastname" value="羅"><br><br>

  <input type="radio" name="gender" value="男">男<br>
  <input type="radio" name="gender" value="女">女<br>


  <input type="submit" value="送出">
</form>
