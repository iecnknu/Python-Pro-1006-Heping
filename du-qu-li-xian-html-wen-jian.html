讀取離線html文件

👀動手做做看：

我們將一步一步帶大家完成離線的html文件讀取。

有一個簡單的html範例文件。

<!DOCTYPE html>
<html>
    <head>
        <style type="text/css">
        .haha{
            color:red
        }
        </style>
        <title>爬蟲範例</title>
        <meta charset="utf-8">
    </head>
    <body>
        <h1>資訊教育中心</h1>
        <p><a href="https://www.python.org/">Python</a>是目前世界上最火熱的程式語言之一，簡潔明瞭的語法與強大豐富的函式庫讓他被運用在各個領域，​<a href="https://www.youtube.com/">YouTube</a> 、<a href="https://www.google.com/">Google</a>、Spotify 都在用它！</p>
        <p class='haha'>學習Python，是進入程式設計世界的最佳途徑！</p>
    </body>
</html>
  1. 創建python檔

    打開sublime3， Ctrl + N 開啟一個新檔案， 緊接著Ctrl + S存檔， 將檔案命名為beautifulSoup4-1.py， 並且將儲存目的地選在"桌面"。(方便待會兒操作)

  2. 匯入模組

    和所有模組一樣，首先使用import將beautifulsoup4匯入程式之中：
    這裡要注意，beautifulsoup的名稱是bs4，我們要從bs4中再匯入BeautifulSoup函式，所以寫法略有不同：

    from bs4 import BeautifulSoup
  3. 放入html範例文件

    請將上述的html範例文件，存入【變數html_doc】之中。

    html_doc = "剛剛的html範例文件"
  4. 讀取文件與解析器(Parser)

    我們使用【BeautifulSoup( )函式】，需要傳入兩個參數。

    BeautifulSoup(htmldoc, "html.parser")

    首先傳入【變數html_doc】，做為要解析的目標。
    再來使用Python內置的html語法解析器【html.parser】，讓BeautifulSoup讀懂變數內的html資料(以html的格式載入)。
    我們可以將透過【BeautifulSoup( )函式】轉化出來的BeautifulSoup物件，貼上soup這個標籤。(也就是存入soup這個變數中)

    soup = BeautifulSoup(htmldoc, "html.parser")

    備註：大家可以試著印出soup這個變數存的內容與其型別，說說看你觀察到什麼？
    (關於物件這個詞，我們將在下學期的物件導向單元詳細介紹。)

  5. 閱讀標籤

    1️⃣在html文件中，一個標籤之下可能含有千千萬萬子標籤。我們可以將這些子標籤稱為【子節點】。
    BeautifulSoup方便之處就在於它有很多方式可以讓我們操作並讀取這些【子節點】。
    如果想要讀取【爬蟲範例】這幾個網頁標題字，觀察文件後發現被存在<title>這個標籤內，我們只要：

    soup.title

    或者想要讀出【資訊教育中心】，觀察文件後發現被存在<h1>這個標籤內，我們只要：

    soup.h1

    就可以取得html文件中，第一個名稱為<h1>或<title>的標籤。

    2️⃣我們也可以

    soup.body.p

    取得html文件中，第一個名稱為body的標籤、裡面第一個名稱為p的子標籤。

  6. BeautifulSoup的類別方法

    在BeautifulSoup中，有三個常用的方法：

    | 名稱 | 描述 | | :--- | :--- | | find( ) | 找到第一個目標標籤。 | | find_all( ) | 找到所有的目標標籤，並以串列的型態回傳。 | | select( ) | CSS選擇器，可透過標籤的名稱、類別和id來定位資料，並以串列的型態回傳。 |

    1️⃣
    剛剛我們透過簡寫的標籤，搜尋到第一個名稱為body的標籤、裡面第一個名稱為p的子標籤。
    其實相等於用兩次的find( )來定位。

    soup.find('body').find('p')

    2️⃣
    但有時我們想要將所有已知的某個標籤一網打盡，就可以用find_all( )來尋找。

    soup.find('body').find_all('a')

    會以串列回傳：

    [<a href="https://www.python.org/">Python</a>, <a href="https://www.youtube.com/">YouTube</a>, <a href="https://www.google.com/">Google</a>]

    3️⃣
    CSS為了讓標籤快速套用某種風格，所以讓使用者可以自訂類別，並輕鬆的將某標籤指定為該類別。
    BeautifulSoup借用了這方便的格式，讓我們使用CSS選擇器，透過標籤的名稱、類別和id來定位資料。
    尋找標籤，直接填入名稱。
    尋找類別，名稱加上【.】。
    尋找id，名稱加上【#】。
    我們想要搜尋類別為haha的標籤，可以寫做：

    soup.select('.haha')

    如果只想要<body>標籤內、類別為haha的子標籤，可以寫做：

    soup.select('body .haha')
  7. 標籤的內容

    標籤內含有許多資訊，比如剛剛想要抓取連結，使用：

    hrefList = soup.find('body').find_all('a')

    來抓取到所有超連結標籤<a>的內容。我們將其存入一個【串列變數hrefList】中。
    但如果我們只需要起始標籤和終止標籤之間的文字，以及存放在href這個屬性中的超連結，該怎麼過濾呢？
    可以寫做：

    #使用迴圈依序提取出hrefList中的內容值
for l in hrefList:
    #可取得標籤中的文字
    print(l.text)
    #可取得標籤的屬性href
    print(l['href'])
