Sublime Text 3 - Package Control

Sublime Text 3 - Package Control

Sublime Text 3 支援【擴充插件 Package】。
就像是先前在Chrome上安裝的JSON Viewer一樣，插件讓原本的軟體增添更多功能，在Sublime上更能簡化軟體工程師的開發流程。

和使用pip來控制Python中的模組一樣，我們使用一個叫做【Package Control】的來管理Sublime Text 3中的插件。

  • 安裝Package Control
  • 檢視Package Control的功能
  • 安裝Package Control
  • 檢視安裝好的Package
🗒️安裝Package Control

  1. 在上方功能清單中選擇： Tools > Command Palette(或者使用快捷鍵 Ctrl + Shift + P)
    【Command Palette命令盤】，可以以指令的方式控制Sublime Text 3。

  2. 在跳出的視窗中輸入Install Package Control，選擇後按Enter開始安裝。

  3. 安裝過程不會有進度條，所以須注意是否跳出安裝完成的提示

  4. 也可以在上方功能清單中選擇：Preferences偏好設定，在下拉清單的最底下看看Package Control是否有出現。

🗒️檢視Package Control的功能
  1. 在上方功能清單中選擇： Tools > Command Palette (或者使用快捷鍵 Ctrl + Shift + P)
  2. 在跳出的視窗中輸入Package Control，右邊就會出現它內部的所有功能。
  3. 幾個常用的功能： 【Install Packages 安裝插件】 【Remove Packages 移除插件】 【Enable Packages 允許插件啟動】 【Disable Packages 不允許插件啟動】 【Upgrade Packages 升級插件】 【List Packages 瀏覽插件清單】
🗒️安裝Package Control

  1. 接續上個步驟，選擇

    Package Control:Install Package

    或者直接在Command Palette輸入

    Install Package

  2. 會跳出一個輸入框，可以搜尋你需要的插件
    也可以一個一個瀏覽，或許會有意想不到的收穫喔！

  3. 輸入名稱，並點擊下方出現的相關插件，即會開始安裝

🗒️檢視安裝好的Package

安裝完插件之後，想要確定是否有確實安裝到Sublime中，可以使用【List Packages】檢查

📝方法1️⃣

  1. 在上方功能清單中選擇： Tools > Command Palette (或者使用快捷鍵 Ctrl + Shift + P)

  2. 在跳出的視窗中輸入

    Package Control:List Packages

    按下Enter後即會導向電腦中安裝Packages的位置。

    C:\Users\使用者名稱\AppData\Roaming\Sublime Text 3\Installed Packages

📝方法2️⃣

  1. 在上方功能清單中選擇：Preferences偏好設定

  2. 在下拉清單的倒數第二個【Package Settings】的次級清單中，是否有出現剛剛安裝的Package名稱。

  3. 使用方法2️⃣可以對插件進行個人化的設定(如果有出現選項的話)。

想要知道更多有關插件的相關新聞或功能介紹，可以上Package Control的官方網站，看看現在時下流行的插件是什麼！
