time - 時間模組

time是python內建的模組，我們可以使用它來處理程式中的日期或時間。

【點我前往官方文件】

  • 使用時間模組
  • time的屬性與函式

🗒️使用時間模組

  1. 創建python檔

    打開sublime3，
    Ctrl + N 開啟一個新檔案，
    緊接著Ctrl + S存檔，
    將檔案命名為time_practice.py，
    並且將儲存目的地選在"桌面"。(方便待會兒操作)

  2. 匯入time模組

    和所有模組一樣，首先使用import將time匯入程式之中：

     import time

  3. 內建函式time( )

    使用內建函式time( )，它會回傳現在的UNIX時間。

    print(time.time())

🗒️time的屬性與函式

  • ⏱️time的屬性

    • daylight

      回應本地時間是否有使用日光節約時間，1表示有，0表示沒有。
      台灣並沒有使用日光節約時間，所以值為0。

      time.daylight

    • timezone

      回傳所在地的時區和UTC相差多少秒，台灣為+8時區，也就是比UTC快了8*60*60=28800秒。

      time.timezone

    • altzone

      回傳所在地的時區和UTC日光節約時間相差多少秒，現在(5/12)正在實行日光標準時間，所以相差為32400秒，比原本的28800秒慢了一小時。

      time.altzone
  • 🕰️time的函式
    • time( )
      #回傳現在的UNIX時間，python是以微秒(0.000001秒)作為單位，所以會回傳一個小數點後六位的浮點數。
time.time()

    • gmtime( [ x ] )

      #傳回目前UTC時間，以time.struct_time結構回傳。
time.gmtime()

      #可以填入選擇性參數 x(UNIX秒數時間)，將其轉換為time.struct_time結構回傳。
time.gmtime(1526043853)

  • time.struct_time結構

屬性 內容
tm_year 西元年分
tm_mon 月份 (1~12)
tm_mday 日 (1~31)
tm_hour 小時 (0~23)
tm_min 分鐘 (0~59)
tm_sec 秒 (0~61(包含潤秒))
tm_wday 星期幾(0~6，從星期一開始)
tm_yday 一年中的第幾天 (1~366(包含閏年))
tm_isdst 是否有日光節約時間 (1有，0沒有，-1表示自動判定)

  • localtime( [ x ] )

    #傳回目前本地時間，以time.struct_time結構回傳。
time.localtime()

    #可以填入選擇性參數 x(UNIX秒數時間)，將其轉換為time.struct_time結構回傳。
time.localtime(1526043853)

  • asctime( [ x ] )

    #以字串型別【星期  月份  日期  時:分:秒 年分】回傳本地時間。
time.asctime()

    #選擇性參數 x(必須是time.struct_time或者tuple)，會回傳指定的時間(年/月/日/時/分/秒/星期幾/一年的第幾天/日光時間)。
time.asctime((2012,12,21,10,11,12,3,0,0))

  • ctime( [ x ] )

    #以字串型別【星期  月份  日期  時:分:秒 年分】回傳本地時間
time.ctime()

    #選擇性參數 x(必須是UNIX秒數)，會回傳指定的時間(年/月/日/時/分/秒/星期幾/一年的第幾天/日光時間)
time.ctime(1526043853))

  • sleep( x )

    #讓python暫停參數x所指定的秒數。
time.sleep(5)

  • strftime( format [, x] )

    將目前時間或選擇性參數 x(必須是time.struct_time或者tuple)，依照format的格式輸出。

  • fortmat的格式符號
格式化符號 內容
%a 星期幾簡寫呈現
%A 星期幾完整呈現
%b 月份簡寫呈現
%B 月份完整呈現
%c 本地適合的日期時間表示法
%d 月份中的第幾天 (1~31)
%H 小時 (0~23，24小時制)
%I 小時 (1~12，12小時制)
%j 一年中的第幾天 (1~366(包含閏年))
%m 月份 (1~12)
%M 分鐘 (0~59)
%p 本地的AM或PM
%S 秒 (0~61(包含潤秒))
%U 一年中的第幾週 (0~53)，星期日為每週的第一天
%w 星期幾(0~6，從星期一開始)
%W 一年中的第幾週(0~53)，星期一為每週的第一天
%x 本地適合的日期表示法
%X 本地適合的時間表示法
%y 西元年的後兩位
%Y 西元年
%z 時區位移(和UTC的時間差)
%Z 時區名稱
%% 顯示出 % 字元 
  ```py
  # 將目前時間以【年/月/日】印出
  print(time.strftime("%Y/%m/%d"))

  # 將目前時間以【年/月/日 時:分:秒】印出
  print(time.strftime("%Y/%m/%d %H:%M:%S"))
  ```
