讀取線上html文件

讀取線上html文件

👀動手做做看：

我們將一步一步帶大家完成線上的html文件讀取。
目標網站：https://beautifulsoup4-1.herokuapp.com/

這裡要再次使用先前介紹過的Requests模組，向網頁伺服器發出HTTP/1.1請求，以獲取完整的html文件。

  1. 創建python檔

    打開sublime3， Ctrl + N 開啟一個新檔案， 緊接著Ctrl + S存檔， 將檔案命名為beautifulSoup4-2.py， 並 且將儲存目的地選在"桌面"。(方便待會兒操作)

  2. 匯入所需模組

    和所有模組一樣，首先使用import將beautifulsoup4匯入程式之中：
    這裡要注意，beautifulsoup的名稱是bs4，我們要從bs4中再匯入BeautifulSoup函式，所以寫法略有不同：

    from bs4 import BeautifulSoup

    接著，使用import將requests匯入程式之中：

    import requests

    接著呼叫requests，搭配向伺服器發送請求的動作方法" get "，

    將我們要向Heroku發出的請求網址當作參數填入，並把結果存入一個變數 webContent 之中：

    webContent = requests.get('https://beautifulsoup4-1.herokuapp.com/')
  3. 設定編碼

    設定webContent的編碼為UTF-8，否則因為HTTP協定，預設編碼會是ISO-8859-1(西歐語言)。

    webContent.encoding = 'UTF-8'
  4. 讀取文件與解析器(Parser)

    我們使用【BeautifulSoup( )函式】，需要傳入兩個參數。
    並將轉化出來的BeautifulSoup物件，貼上soup這個標籤。

    soup = BeautifulSoup(webContent.text, "html.parser")

    首先傳入【變數webContent】，做為要解析的目標。
    webContent是一個Requests物件，我們需要取得其text屬性值，才會是我們需要的html文字檔。
    再來使用Python內置的html語法解析器【html.parser】，讓BeautifulSoup讀懂變數內的html資料(以html的格式載入)。
