# PyPi

![](https://drive.google.com/uc?export=download&id=1pXcBll6-IZUHtcTorMTa4Uewlyzj4zE8)

**Pypi，**\(Python Package Index，Python套件索引\)，是一個為Python打造的線上程式倉庫。  
**PyPI **可以幫助我們搜尋和安裝Python社區開發和共享的軟體。  
套件的作者也使用 **PyPI **來發布他們開發的軟體。

[點我前往](https://pypi.org/)
