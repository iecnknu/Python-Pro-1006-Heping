Turtle模組指令(函式)

Turtle模組指令(函式)

Turtle模組的詳細指令，可以到官網查詢。
這裡列出幾個常用到的指令，並簡單地替各位同學分類。

  • 畫筆動作指令
  • 畫筆控制指令
  • 全局控制指令
  • 其他指令
  • 畫布指令
  • 取得畫筆狀態指令
1️⃣畫筆動作指令
指令 說明
home() 將turtle移動到起點（0,0）和向東
speed(speed) 畫筆繪製的速度範圍[0,10]整數
forward(distance) 向當前畫筆方向移動distance像素長
backward(distance) 向當前畫筆相反方向移動distance像素長度
right(degree) 順時針移動degree°
left(degree) 逆時針移動degree°
pendown() 移動時繪製圖形放下筆,,默認繪製
penup() 移動時不繪製圖形,提起筆，用於另起一個地方繪製時用
circle(r,extent,step) 繪製一個指定半徑r、弧度範圍extent、階數step(正多邊形)的圓
dot(diameter,color) 繪製一個指定直徑和顏色的圓
goto( x , y ) 移動畫筆到指定的座標( x , y )
sety( y ) 移動畫筆到指定的y座標
setx( x ) 移動畫筆到指定的x座標
setheading(degree) 轉動畫筆到指定的角度degree。
2️⃣畫筆控制指令
指令 說明
pencolor() 畫筆顏色
pensize(width) 畫筆寬度（繪製圖形時的寬度）
shape( name ) 設定畫筆的形狀。有這些形狀可以填入【arrow, turtle, circle, square, triangle, classic】。如果有在screen註冊gif檔，也可以直接指定使用。
shapesize( stretch_wid, stretch_len, outline ) 設定畫筆的形狀細項，
color(color1, color2) 同時設置pencolor=color1, fillcolor=color2
filling() 返回當前是否在填充狀態
begin_fill() 準備開始填充圖形
end_fill() 填充完成
hideturtle() 隱藏畫筆的turtle形狀
showturtle() 顯示畫筆的turtle形狀
3️⃣全局控制指令
指令 說明
clear() 清空turtle窗口，但是turtle的位置和狀態不會改變
reset() 清空窗口，重置turtle狀態為起始狀態
undo() 取消最後一個圖的操作，返回上一步。
isvisible() 返回當前turtle是否可見
stamp() 蓋下筆觸
write(s,move,align,[font=("font-name",font_size,"font_type")]) 寫文本，s為文本內容；font是字體的參數，分別為字體名稱，大小和類型；font為可選項，font參數也是可選項(字體名稱,字體大小,字體外觀)；move，設定撰寫時畫筆是否移動；align，設定文字的排版，可以設定left靠左、center置中、right靠右

推薦使用Digital-7 Mono字體

4️⃣其他指令
指令 說明
Turtle() 建立畫筆。(建立turtle物件)
mainloop()或done() 啟動事件循環-調用Tkinter的mainloop函數。必須是烏龜圖形程序中的最後一個語句。(可以讓畫面停留)
screensize(canvwidth, canvheight, bg) 設定視窗大小，同時設置canvwidth螢幕寬度、canvheight螢幕高度、bg背景顏色
5️⃣畫布指令
指令 說明
Screen() 建立畫布。(建立turtle.screen物件)
setup( width , height) 設定畫布的寬高。width 是寬、height是高。可以設定為占用使用者螢幕的某個比例，也可以設定確切的像素。
title( name ) 設定視窗的標題。name是參數。
bgcolor( color ) 設定背景色彩。color可以填入顏色。
addshape( path) 匯入gif檔，可作為畫筆的形狀。path為要匯入的檔案的路徑，可填入相對路徑與絕對路徑。
ontimer(fun, time) 設定一個計時器，在time(單位為毫秒)後，呼叫function。
onkey(fun, key) 綁定按鍵key與fun函式。當按鍵被觸發時，呼叫function。
listen() 讓電腦聚焦在畫布上，讓onkey()綁定的按鍵與函式可以被觸發。
6️⃣取得畫筆狀態指令
指令 說明
position() 取得畫筆的( x , y )座標。
towards( x , y ) 取得畫筆與目標( x , y )之間的夾角。
xcor() 取得畫筆的x座標。
ycor() 取得畫筆的y座標。
heading() 取得畫筆目前指的方向。
distance( x , y ) 取得畫筆與目標( x , y )之間的距離。
