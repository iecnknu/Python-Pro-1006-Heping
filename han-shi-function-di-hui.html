遞迴定義

遞迴是指【函式呼叫函式本身】，在一個函式Function1( )中再呼叫函式Function1( )。

它的核心就是把大問題切割成小問題依序解決。

遞迴像是【判斷式】和【迴圈】的結合體，補足了if判斷式無法迴圈的問題，也解決了迴圈不能依次判斷的窘境。

若使用得當，會發現它的可讀性高於一般迴圈很多！

撰寫遞迴函式最重要的步驟就是能拆解問題，列出問題的起始條件及終止條件，並依序在函式中設定判斷式。

範例：

若我們要寫一個【for迴圈】計算階乘，也就是當使用者輸入正整數n時，會計算出n!=n*(n-1)*(n-2)...2*1

遞迴範例-1

num = int(input('請輸入正整數'))
result = 1
if num < 0:
    result = '請輸入大於0的正整數'
else:
    for i in range(1,num+1):
        result = result*i
print(result)

那讓我們看看用遞迴函式的話要怎麼做，首先我們先列出問題的起始條件及終止條件，

計算階乘的函式為factorial(n)：

  • 當n = 0，0! = 1 ➡️ factorial(0) = 1
  • 當n > 0，n!=n(n-1)! ➡️ factorial(n) = n*factorial(n-1)
  • 當n < 0，提示使用者輸入大於0的數字

遞迴範例-2

def factorial(n):
  if n == 0:
    return 1
  elif n > 0:
    return n*factorial(n-1)
  else:
    return '輸入大於0的數'

print(factorial(int(input('輸入正整數'))))

計算斐波那契數列的函式為fibonacci(n)：

  • 當n = 1 ➡️ fibonacci(1) = 1
  • 當n = 2 ➡️ fibonacci(2) = 1
  • 當n > 2，fibonacci(n) = fibonacci(n-1) + fibonacci(n-2)
